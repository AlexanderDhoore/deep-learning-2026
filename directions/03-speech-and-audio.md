# Domain 6: Speech and Audio

## Good fit if you like

- voice
- audio interfaces
- transcription
- interactive demos

## Typical project ideas

- speech-to-text tool
- text-to-speech assistant
- voice note summarizer
- sound classifier
- voice-enabled chatbot
- music generation demo
- voice adaptation demo

## Strong starter tools

- Whisper
- Transformers
- torchaudio
- Kokoro or another TTS library
- Gradio

## Good advice

- use existing pretrained models
- keep audio handling simple
- test with a few clear use cases
- this domain can use Path 1, Path 2, or Path 3
