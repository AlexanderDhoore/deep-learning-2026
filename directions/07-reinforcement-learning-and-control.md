# Direction 7: Reinforcement Learning and Control

## Good fit if you like

- simulation
- control
- robotics
- decision making

## Typical project ideas

- train an agent in a simple environment
- compare RL algorithms on one task
- make an interactive policy demo

## Strong starter tools

- Gymnasium
- Stable-Baselines3
- TorchRL
- PyTorch

## Good advice

- keep the environment simple
- do not overscope
- this direction is interesting but riskier than most others
- Stable-Baselines3 is the safest starting point for most projects
