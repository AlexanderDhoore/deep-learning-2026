# Domain 1: Computer Vision

## Good fit if you like

- images
- video
- cameras
- visual demos
- practical engineering

## Typical project ideas

- object detector for a chosen domain
- image classifier
- defect detection demo
- segmentation app
- depth estimation demo
- keypoint detection demo
- people or object counter
- pose estimation demo

## Strong starter tools

- PyTorch
- torchvision
- timm
- Ultralytics YOLO
- RF-DETR
- OpenCV
- Gradio / Streamlit / FastAPI

## Good advice

- start from a pretrained model
- keep the dataset small
- focus on one clear task
- make the output visual
- this is an established field, but still extremely powerful in practice
