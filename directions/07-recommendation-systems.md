# Direction 7: Recommendation Systems

## Good fit if you like

- personalization
- ranking
- media or product discovery
- practical AI products

## Typical project ideas

- music or media recommender
- product recommendation demo
- similar-item recommender
- recommendation app based on embeddings
- content-based recommendation tool

## Strong starter tools

- Sentence Transformers
- PyTorch for custom ranking experiments
- vector search libraries or databases
- Gradio / Streamlit / FastAPI

## Good advice

- start with a simple content-based approach
- keep the recommendation logic understandable
- focus on one dataset and one clear user experience
