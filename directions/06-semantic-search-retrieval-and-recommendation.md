# Direction 6: Semantic Search, Retrieval, and Recommendation

## Good fit if you like

- search
- information retrieval
- recommendation systems
- text or media discovery
- practical AI products

## Typical project ideas

- semantic search through notes or documents
- retrieval-augmented chatbot
- music or media recommender
- similar-item finder
- recommendation app based on embeddings

## Strong starter tools

- Sentence Transformers
- Transformers
- vector search libraries or databases
- Gradio / Streamlit / FastAPI

## Good advice

- start with embeddings and cosine similarity
- keep the recommendation logic understandable
- focus on one dataset and one clear user task
