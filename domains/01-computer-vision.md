# Domain 1: Computer Vision

Computer vision is one of the clearest and most practical deep-learning domains.

## What This Domain Is

This domain is about using AI on images and video.

Typical tasks include:

- image classification
- object detection
- segmentation
- depth estimation
- keypoint detection

## Why You Might Choose It

- it is visual and easy to demo
- many pretrained models already exist
- it works well on the hardware you have
- it is an established field, but still extremely powerful in practice
- it solves many real industrial problems very well

## Example Applications

- object detector for a chosen domain
- image classifier
- defect detection demo
- segmentation app
- depth estimation demo
- keypoint detection demo
- people or object counter
- pose estimation demo

## Which Technical Paths Fit Best

- Path 1 if you want to train or retrain your own model
- Path 2 if you want to build around an existing pretrained model

## Recommended Frameworks to Start With

- PyTorch
- torchvision
- timm
- Ultralytics YOLO
- RF-DETR
- OpenCV

## Good Advice Before You Begin

- start from a pretrained model
- keep the dataset small
- focus on one clear task
- make the output visual
