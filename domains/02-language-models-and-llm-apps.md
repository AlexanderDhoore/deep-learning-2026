# Domain 2: Language Models and Text Systems

This domain is about building AI systems that work mainly with text.

## What This Domain Is

Typical tasks include:

- chat systems
- question answering
- summarization
- classification
- tool calling
- text workflows

## Why You Might Choose It

- you already touched this domain in the generative AI workshop
- it is easy to prototype quickly
- it is easy to combine with a frontend
- many modern AI products are built in this space

## Example Applications

- chatbot for a narrow domain
- summarizer
- classifier for messages or documents
- question-answering tool
- tool-calling assistant

## Which Technical Paths Fit Best

- Path 2 if you want to build with pretrained models
- Path 3 if you want to adapt or fine-tune a large model
- this domain usually does not start with full training from scratch

## Recommended Frameworks to Start With

- Ollama
- Hugging Face Transformers
- Sentence Transformers
- Gradio
- FastAPI

## Official Docs and Starting Points

- Ollama: <https://ollama.com/>
- Hugging Face Transformers: <https://huggingface.co/docs/transformers/en/index>
- Sentence Transformers: <https://www.sbert.net/>
- Gradio: <https://www.gradio.app/docs>
- FastAPI: <https://fastapi.tiangolo.com/>

## Good Advice Before You Begin

- narrow the scope
- avoid trying to build a general chatbot for everything
- give the app one clear purpose
- this domain usually relies on pretrained or fine-tuned models
- tool calling is powerful enough to become a whole project by itself
